## 📋 部署和使用指南

### 系统要求
- **操作系统**: Windows 10+, Linux, macOS
- **Python版本**: Python 3.8+
- **内存**: 最小4GB，推荐8GB
- **存储**: 最小10GB可用空间
- **网络**: 稳定的互联网连接

### 安装步骤
1. **克隆项目**: `git clone <repository-url>`
2. **安装依赖**: `pip install -r requirements.txt`
3. **配置环境**: 设置环境变量
4. **验证配置**: 运行配置验证脚本
5. **启动演示**: 运行演示脚本

### 配置说明
- **FFmpeg**: 安装FFmpeg并添加到系统PATH
- **扩散模型**: 配置xingzi/diffuEraser模型
- **输出路径**: 配置输出文件路径
- **日志级别**: 配置日志级别

### 使用方法
1. **视频字幕压制**: 使用SubtitlePressing类
2. **字幕无痕擦除**: 使用SubtitleErasure类
3. **配置管理**: 使用ConfigManager类
4. **演示脚本**: 运行demo_subtitle_pressing_erasure.py
5. **测试脚本**: 运行test_subtitle_features.py

### 故障排除
- **FFmpeg未找到**: 安装FFmpeg并添加到PATH
- **模型不可用**: 检查模型可用性状态
- **内存不足**: 增加内存或减少并发数
- **处理速度慢**: 检查网络速度和服务器负载
