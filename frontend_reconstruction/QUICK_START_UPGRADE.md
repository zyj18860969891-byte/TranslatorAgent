# Translator Agent 详情页升级 - 快速启动指南

## 🚀 快速开始

### 1. 启动开发服务器
```bash
cd frontend_reconstruction
npm run dev
```

### 2. 访问应用
打开浏览器访问 `http://localhost:5173`

### 3. 使用详情页功能

#### 步骤1: 选择专业模块
- 点击主页的"专业视频翻译"、"字幕提取"等卡片
- 系统会自动创建 Sub-Agent 会话

#### 步骤2: 上传文件
- **拖拽上传**: 将视频或字幕文件拖拽到上传区域
- **点击上传**: 点击"选择文件"按钮选择文件
- **支持格式**: 视频(.mp4, .avi, .mov)、字幕(.srt)、文本(.txt)

#### 步骤3: 输入自然语言指令
在指令输入框中输入任务描述，例如：
- "将视频翻译成日语，保持情感基调一致"
- "提取视频中的英文字幕"
- "翻译字幕文件为中文，保持时间轴对齐"

#### 步骤4: 发送任务
- 点击"发送"按钮
- 系统会自动开始处理并显示实时进度

#### 步骤5: 查看进度
- **实时进度条**: 显示整体处理进度
- **步骤进度**: 显示上传、处理、翻译、完成等步骤
- **文件统计**: 显示已上传、已处理、失败的文件数量
- **记忆层统计**: 显示对话、结果、中间数据的数量

#### 步骤6: 查看结果
- 在右侧文件区查看处理完成的文件
- 点击下载按钮下载结果文件
- 文件状态会实时更新

## 🔧 故障排除

### 问题1: 无法创建任务会话
**症状**: 点击模块后没有反应
**解决**: 
1. 检查浏览器控制台是否有错误
2. 确认 localStorage 可用
3. 刷新页面重试

### 问题2: 文件上传失败
**症状**: 文件上传后没有显示
**解决**:
1. 检查文件格式是否支持
2. 检查文件大小是否超过限制
3. 查看浏览器控制台错误信息

### 问题3: 进度条不更新
**症状**: 进度条一直显示 0%
**解决**:
1. 检查文件系统状态机是否初始化
2. 查看网络请求是否正常
3. 刷新页面重试

### 问题4: 编译错误
**症状**: `npm run build` 失败
**解决**:
1. 检查 TypeScript 版本
2. 清理 node_modules: `rm -rf node_modules`
3. 重新安装依赖: `npm install`
