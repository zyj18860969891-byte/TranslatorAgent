# OpenManus Translator Agent - 快速使用指南

## 启动应用

### 开发模式
```bash
cd D:\MultiMode\TranslatorAgent\frontend_reconstruction
npm run dev
```

### 生产模式
```bash
npm run build
# 然后将 dist/ 目录部署到静态文件服务器
```

## 使用流程

### 1. 访问主页
打开浏览器访问 `http://localhost:3000/`，您将看到功能选择页面。

### 2. 选择功能
页面展示6个功能卡片：
- **文本翻译** - 支持多语言文本翻译
- **视频处理** - 视频字幕提取、视觉分析
- **字幕编辑** - 字幕预览、时间轴编辑
- **快速处理** - 批处理优化
- **安全可靠** - 本地处理保护隐私
- **多语言支持** - 支持多种语言

点击任意一个功能卡片（如"视频处理"）将自动跳转到交互式详情页。

### 3. 交互式详情页界面

详情页采用三栏式布局：

#### 左侧 - 历史任务导航
- 显示最近的历史任务列表
- 点击任务可加载相关文件
- 显示任务状态（完成/处理中）
- 显示任务时间戳

#### 中间 - 对话交互区
- **聊天消息区**：显示用户和AI的对话历史
- **文件上传区**：点击"上传文件"按钮选择文件
- **输入区**：输入指令或消息，按Enter发送
- **处理进度**：显示文件处理进度条

#### 右侧 - 实时任务文件区
- **文件列表**：显示所有上传和处理的文件
- **状态过滤**：可按类型筛选（全部/视频/字幕/文本/结果）
- **统计信息**：显示文件总数、完成数、处理中数
- **文件操作**：支持选择、下载、删除文件

### 4. 上传文件

1. 点击中间区域的"上传文件"按钮
2. 选择要处理的文件（支持视频、字幕、文本文件）
3. 文件会自动添加到右侧文件区
4. 上传完成后自动开始处理

### 5. 对话交互

在输入框中输入指令，例如：
- "翻译成日语"
- "提取字幕"
- "处理这个视频文件"
- "保留原片的情感"

按Enter或点击"发送"按钮，AI会：
1. 理解您的指令
2. 处理上传的文件
3. 在聊天区回复处理结果
4. 在右侧文件区更新文件状态

### 6. 查看处理结果

- **聊天区**：查看AI的回复和处理结果
- **文件区**：查看文件状态变化
  - ⏳ 待处理 → 🔵 处理中 → ✅ 完成
- **下载结果**：点击完成状态文件旁的下载按钮

### 7. 使用历史任务

1. 在左侧历史任务列表中点击任务
2. 系统自动加载该任务的文件
3. 可以继续处理或查看结果

## 示例场景

### 场景1：视频翻译
1. 点击主页的"视频处理"卡片
2. 上传视频文件（如：`example.mp4`）
3. 输入指令："翻译成中文，保留原片的情感"
4. AI开始处理，显示进度条
5. 处理完成后，在右侧文件区下载翻译结果

### 场景2：字幕提取
1. 点击主页的"字幕编辑"卡片
2. 上传视频文件
3. 输入指令："提取字幕并翻译成英文"
4. AI提取字幕并翻译
5. 在右侧文件区下载SRT字幕文件

### 场景3：文本翻译
1. 点击主页的"文本翻译"卡片
2. 上传文本文件（如：`document.txt`）
3. 输入指令："翻译成日语，保持专业术语"
4. AI翻译文本
5. 在右侧文件区下载翻译后的文本

## 文件状态说明

| 状态 | 图标 | 含义 |
|------|------|------|
| 待处理 | ⏳ | 文件已上传，等待处理 |
| 处理中 | 🔵 | 文件正在处理中 |
| 完成 | ✅ | 文件处理完成，可下载 |
| 错误 | ❌ | 处理失败，需要重试 |

## 快捷操作

- **Enter**：发送消息（Shift+Enter换行）
- **上传文件**：支持多选文件
- **刷新文件列表**：点击右侧刷新按钮
- **返回主页**：点击详情页右上角"返回主页"按钮

## 注意事项

1. **文件类型支持**
   - 视频：`.mp4`, `.avi`, `.mov`, `.mkv` 等
   - 字幕：`.srt`, `.vtt` 等
   - 文本：`.txt`, `.md` 等

2. **浏览器兼容性**
   - 推荐使用 Chrome、Edge、Firefox 最新版
   - 支持现代浏览器的文件API

3. **性能建议**
   - 大文件上传可能需要较长时间
   - 建议分批处理多个文件
   - 保持浏览器标签页活跃以获得最佳体验

4. **数据安全**
   - 所有处理在本地进行（根据架构设计）
   - 文件不会上传到外部服务器
   - 处理完成后可手动删除文件

## 故障排除

### 问题：页面无法加载
**解决方案**：
- 检查开发服务器是否运行
- 确认端口3000未被占用
- 查看浏览器控制台错误信息

### 问题：文件上传失败
**解决方案**：
- 检查文件大小限制
- 确认文件类型是否支持
- 尝试重新选择文件

### 问题：处理进度卡住
**解决方案**：
- 刷新页面重新上传
- 检查浏览器控制台错误
- 尝试较小的文件

### 问题：无法下载结果
**解决方案**：
- 确认文件状态为"完成"
- 检查浏览器下载设置
- 尝试使用其他浏览器
